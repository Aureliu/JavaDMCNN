training HMM Name Tagger

The training procedure for the Jet HMM Name tagger is invoked by
xjet Jet.HMM.HMMNameTagger state-file model-file order directory1 filelist1 [directory2 filelist2] ...

with 2n+3 arguments, as follows
state-file
a file defining states of the HMM (input)
model-file
a file containing trained HMM model (output)
order
either "unigram" or "bigram", indicating what emission probabilities should be used
directoryi
directory containing ne-tagged files
filelisti
list of ne-tagged files (relative to directoryi)
